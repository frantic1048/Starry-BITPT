#Starry-BITPT

极速之星全站夜空风格的混乱邪恶的 Stylish 样式。

##使用事项

为了保证最佳风味，请务必将代码#49行处的URL中的 URL 替换为自己的极速空间中的大头像的 URL。

    /*-------------头像-----------*/
    #um .avt.y a {
      background-image: url(http://bitpt.cn/ucenter/data/avatar/.......jpg); }

    /*----------------------------*/
